# 🍎 FRUID: AI-driven Intelligent E-commerce Ecosystem

FRUID is a modern e-commerce platform built using PHP, Machine Learning, and Deep Learning. It combines a typical shopping experience with Artificial Intelligence, Image Classification, Cyberbullying Detection, and Real-time Logistics. It’s not just a regular shop. I have turned the shops into smart spaces using AI automation. This makes managing the store easier and improves the shopping experience for everyone.

## Features

- **AI Image Search:** Upload an image to directly find products in the store.  
- **AI Image Classification:** Admin can upload an image to automatically identify unknown fruits or vegetables.  
- **Cyberbullying Detection:** Detect harmful text in the chat system and in comments or review sections (Cyberbully_ML).  
- **Smoothie Builder:** Interactive feature that lets customers easily customize smoothies and receive recipe suggestions.  
- **AI Nutrition Insight Planner:** Provides AI-generated nutrition plans with product recommendations based on user information.  
- **User Management:** Secure membership system with email verification and password recovery.  
- **AI Voice Assistant Feature:** Customers can use hands-free voice commands to navigate and search the fruit store.  
- **AI Info Generator:** Automatically generates high-quality product descriptions, reducing manual work for the admin.
